A) HTML Structure - top

This theme is a responsive layout with 12 columns (Bootstrap CSS). All of the information within the main content area is nested within a div with an id of "page-wrapper". Inside of any "row" class, you can use 1 - 12 colums. (col-lg-1, col-lg-5, col-lg-12)

B) CSS Files and Structure - top

I'm using 2 CSS files in this theme connected to main (style.css) file.

  • bootstrap.css > Responsive layout and main contents such as buttons, navigations, etc.
  • style.css > Main CSS code files. You will need to use this file mostly to perform the changes

The main file (style.css) contains all of the specific stylings for the page. The file is separated into sections using:

	/*----------------------------
			.-- Imports
			-----------------------------*/

			some code

			/*----------------------------
			.-- Resets
			-----------------------------*/
			
			some code
			
			/*----------------------------
			.-- General
			-----------------------------*/
			
			some code
			
			/*----------------------------
			.-- Navigation
			-----------------------------*/
			
			some code
			
			etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

If you would like to edit the navigation, find the following section in the style sheet:

	ul.nav li.dropdown:hover > ul.dropdown-menu{
					change style here
					}

C) JavaScript - top

This theme imports 9 Javascript files.

  1. settings.js

    Most of the scripts are here if you need to change anything. But due to the dynamic structure, you won't need to change any of these codes but some settings such as Google Map, Contact, Twitter, please, see the Settings section.

  2. twitterconfig

    You need to edit this file to change the twitter settings

  3. bootstrap.js
  4. countdown.js

  6. jquery.js

  8. twitterfetcher.js
  9. validate.js

D) Settings - top

Images



  • Open the index/blog.html and replace the url with your images.

Changing Colors



  • Open custom.css file from the folder _css with any Notepad software. Go to "Replace". Enter the first and second color code, click replace all. Save the file.

Countdown



  • Open settings.js file from the folder _js with any Notepad software. Change the dates and save it.

Dropdown menu

Use the example below for dropdown menu

Twitter Settings



  • Open twitterconfig.js file from the folder _js with any Notepad software.
  • Replace the ID with yours
    • Create Twitter ID
    • Go to www.twitter.com and sign in as normal, go to your settings page.
    • Go to "Widgets" on the left hand side.
    • Create a new widget for what you need eg "user time line" or "search" etc.
    • Feel free to check "exclude replies" if you don't want replies in results.
    • Now go back to settings page, and then go back to widgets page and you should see the widget you just created. Click edit.
    • Look at the URL in your web browser, you will see a long number like this: '45735908357048478' Use this as your ID above instead!
